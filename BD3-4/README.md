# Banco de Dados III e IV

**Curso:** Técnico em Informática  
**Disciplina:** Banco de Dados III e IV  
**Período Letivo:** 3º Série  


## Ementa
Implementação de Banco de Dados. Uso aperfeiçoado do sistema de consulta.

## Sistema Gerenciador de Banco de Dados (SGBD)
**PostgreSQL** será o SGBD utilizado neste curso. Escolhemos PostgreSQL por sua forte aderência no mercado de tecnologia, robustez comprovada, segurança avançada e, além disso, é totalmente **open source** — permitindo aprendizado transparente da tecnologia.

## Competências / Habilidades
- Implementar banco de dados relacional utilizando um Sistema Gerenciador de Banco de Dados.
- Otimizar a linguagem de consulta estruturada como forma de informação para tomada de decisão.
- Utilizar sistemas de gerenciamento para bancos de dados.
- Executar linguagem de consulta estruturada objetivando melhor desempenho.
- Compilar relatórios analíticos a partir dos dados coletados.

## Bases Científico-Tecnológicas
- Implementação de banco de dados: criação e exclusão de bancos de dados.
- Comandos SQL (DDL, DML, DQL): conceitos e utilização.
- Blocos de linguagem de consulta estruturada (SQL), tratamento de exceções e funções.
